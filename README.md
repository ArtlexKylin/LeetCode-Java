# 首页目录/文章标题 排版规则
1. 分门别类的根据解法难度对题目进行划分
2. 对题目必要度(性价比)划分（0~3星）
3. 对一些有趣解法的标记
4. 通过直接把测试用例写在程序里的方法不会记载在题解中
